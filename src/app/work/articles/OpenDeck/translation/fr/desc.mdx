L'Open Deck est un écran géant semi-circulaire qui possède 8 rétroprojecteurs 4K.

Cette application permet de controller chaque projecteur,
ainsi que le système audio et de climatisation.
Le tout depuis son téléphone portable.

Pour ce faire, nous avons créé un serveur REST en Python que nous accédons par WebSockets.

Le design est responsive : il s'adapte à toutes les tailles et orientations d'écran.
