# HandsOnWebGL

À l'occasion d'une rencontre avec un groupe d'étudiants, nous avons décidé de préparer un petit exercice interactif pour expliquer ce que faisait
l'équipe de visualisation du Blue Brain Project.

C'était une session d'une heure alors nous avons décidé de nous concentrer sur un scénario plutôt simple :
coder une animation de propagation d'un signal électrique le long des dendrites d'un neurone.

Le code GLSL est compilé au fil de l'eau et le résult apparaît immédiatement.
Les étudiants peuvent alors expérimenter et apprendre plus rapidement de leurs erreurs.

[Essayez cette démo de 3 minutes](apps/hands-on-webgl/index.html). Mieux vaut un écran pas trop petit.
